01

Goalivo란?

Goalivo는 목표 달성에 특화된 타임라인 캘린더입니다. 단순히 일정을 기록하는 것을 넘어, 목표를 설정하면 AI가 역산해서 마일스톤과 일정을 자동으로 만들어 줍니다.

🎯 목표 중심 설계

D-Day 카운트다운, 주간 시간 할당, 진행률 추적까지 목표 달성에 필요한 모든 것

🤖 AI 역산 & 리포트

목표를 입력하면 AI가 마일스톤을 생성하고, 현재 수준에 맞는 맞춤형 꿀팁 리포트 제공

📅 8가지 뷰 모드

Day부터 10 Years까지 — 하루 일과부터 인생 계획까지 한눈에

⏰ 타임테이블 & 루틴

10분 단위 시간 블록과 반복 루틴으로 하루를 체계적으로 관리

🎨 5가지 테마 스킨

Midnight, Keepsake, Archivist, Aurora, Obsidian — 취향에 맞는 디자인

🌐 클라우드 동기화

어디서든 로그인하면 모든 기기에서 데이터 동기화

04

목표 관리 차별화 기능

Goalivo의 핵심은 목표 중심의 캘린더입니다. 목표를 만들면 D-Day 카운트다운, 진행률, 주간 시간 할당이 자동으로 계산됩니다.

목표 역산 기능 Goalivo Only

일반 캘린더는 일정을 "기록"하지만, Goalivo는 목표에서 역산합니다. "언제까지 달성할래?"를 입력하면 → 지금부터 무엇을 해야 하는지 자동으로 계산합니다.

🔄
역산이란? 목표 기한과 필요한 총 시간을 입력하면, Goalivo가 남은 기간을 주 단위로 나눠 하루에 얼마나 투입해야 하는지 자동으로 계산합니다. 마일스톤이 지연되면 자동 리밸런싱으로 나머지 일정이 재조정됩니다.

06

마일스톤 & 태스크 Goalivo Only

목표를 달성 가능한 작은 단계로 쪼갭니다. AI가 자동 생성하거나 직접 만들 수 있습니다.

🤖 AI 자동 생성

목표를 저장하면 AI가 난이도·기간을 고려해 마일스톤을 자동으로 설계합니다.

✏️ 수동 생성 & 편집

직접 마일스톤을 추가·수정하고 순서를 드래그로 변경할 수 있습니다.

⚡ 자동 리밸런싱

마일스톤이 지연되면 남은 일정을 자동으로 재조정해 목표일을 유지합니다.

📊 Quarter/Year 뷰 연동

마일스톤이 분기 뷰와 연간 뷰에 타임라인으로 시각화되어 한눈에 파악 가능합니다.

🔗
Quarter & Year 뷰에서 확인: 상단 핵심 기능 하이라이트에서 실제 마일스톤 뷰 화면을 미리 확인할 수 있습니다. 목표별 진행률이 컬러 바로 표시됩니다.

12

비전보드 차별화 기능

달성하고 싶은 목표의 이미지를 최대 6장까지 업로드하고, 언제든 확인하며 동기부여를 유지하세요.

📷 이미지 업로드

갤러리 또는 카메라에서 목표 이미지를 최대 6장 업로드. 언제든 교체 가능합니다.

⏰ 시간별 팝업 알림

지정한 시간에 비전 이미지가 자동으로 팝업으로 나타납니다. 하루에 여러 시간 설정 가능.

🔄 이미지 순환 표시

등록된 이미지가 팝업마다 순환하여 매일 새로운 비전으로 동기부여를 유지합니다.

📊 목표 연동 팝업

팝업에 D-Day 카운트다운과 진행률이 함께 표시되어 현재 목표 상황을 즉시 확인합니다.

설정 메뉴 → 비전보드 열기

앱 우측 하단 메뉴에서 비전보드 탭을 선택합니다.

+ 이미지 추가 버튼 탭

갤러리에서 목표를 시각화하는 이미지를 선택합니다. (내 집, 여행지, 목표 체형, 커리어 등)

⏰ 팝업 설정 탭

팝업을 받고 싶은 시간을 설정합니다. (예: 오전 7:30, 오후 12:00)

지정 시간에 비전 팝업 확인

설정한 시간이 되면 비전 이미지가 팝업으로 나타나며 오늘 할 일 바로가기 버튼도 제공됩니다.

💡
추천: 기상 직후와 점심 시간에 팝업 알림을 설정하면 하루 두 번 목표 비전을 확인하며 집중력을 유지할 수 있습니다. 상단 핵심 기능 하이라이트에서 실제 화면을 확인해 보세요.

20

FAQ

Q. 무료인가요?

네, Goalivo의 핵심 기능은 모두 무료입니다.

Q. 모바일에서도 사용할 수 있나요?

네! 모바일 브라우저에서 완벽하게 동작합니다.

Q. 데이터가 안전한가요?

데이터는 로컬 저장소와 클라우드에 이중 저장됩니다. JSON 내보내기로 추가 백업도 가능합니다.

Q. 인터넷 없이도 사용할 수 있나요?

기본 기능은 오프라인에서도 사용 가능합니다.
